vCal file for university teaching weeks

Want to have the University's teaching weeks in your calendaring application? Use this vCal file. Simply import it into your application of choice (such as Outlook on Windows or Calendar.app on macOS). It contains all-day events every Monday which simply name the teaching week (e.g., S1, 22, etc.).

Want to know how I made this? I had to copy the information in this PDF file, and parse that pasted content using awk and some other Unix utilities. Yuck.
